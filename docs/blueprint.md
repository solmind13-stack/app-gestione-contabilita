## Core Features:

- Bank Movement Tracking: Register and monitor all bank transactions for LNC and STG. Supports filtering by year and company.
- Fiscal Deadline Management: Manage tax deadlines and payments with automatic alerts and notifications.
- AI-Powered Insights and Suggestions: Utilize Gemini to provide intelligent analysis, suggestions, and financial forecasts for improved decision-making. It acts as a tool for data processing.
- Automated Narrative Reports: Generate automated narrative reports summarizing financial data and insights.
- Cash Flow Forecasting: Predict future cash flow with multiple scenarios, including optimistic, realistic, and pessimistic views.
- Interactive Dashboards: Visualize key performance indicators (KPIs) and financial data through interactive dashboards and charts.
- User Authentication and Roles: Implement Firebase Authentication with email and password, managing user roles (admin, lnc, stg) to control data access.

## Style Guidelines:

- Primary color: Dark blue (#1a365d) for a professional and trustworthy feel.
- Background color: Light gray (#f7fafc) for a clean and modern look.
- Accent color: Green (#38a169) for positive indicators and calls to action. To visually separate data for the two companies, #4472C4 will also be used, to signify LNC
- Body and headline font: 'Inter', a grotesque-style sans-serif, for a modern and neutral aesthetic.
- Code font: 'Source Code Pro' for displaying numerical data.
- Use a consistent set of icons from a library like Font Awesome or Material Icons for navigation and actions.
- Maintain a clean and organized layout with clear sections and spacing, inspired by SaaS dashboards like Stripe or QuickBooks.
